# Accessible Component Library

A small set of accessible UI components built with React and Tailwind CSS.

## Goal

This project was created to demonstrate how simple components can be designed with clarity, semantic structure and accessibility in mind.

## Components

* Navigation with mega menu
* Dropdown menu button
* Search bar
* Article card
* Footer

## Accessibility considerations

* Semantic HTML (nav, main, article, footer)
* Keyboard navigation (Tab, Escape)
* Visible focus states
* Accessible labels and roles
* Clear interaction patterns

## Tech

* React
* Tailwind CSS
* Vite
